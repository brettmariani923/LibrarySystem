Developed a fully functional RESTful API for managing a digital library using Clean Architecture principles in .NET 8. Implemented Entity Framework Core for data access and persistence. The system supports CRUD operations for books, with layers divided into Domain, Application, and Data to ensure maintainability, scalability, and testability.
